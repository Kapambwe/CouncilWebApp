# Lusaka City Council - Integrated Development Plans Sample Data

This directory contains sample JSON data files for the Lusaka City Council's Integrated Development Plans component. All data is realistic and contextually appropriate for Zambian local government strategic planning and development.

## Overview

This collection includes **12 JSON files**, each containing **30 sample data entries** for different aspects of integrated development planning.

## Data Context

- **Location**: Lusaka City Council, Zambia
- **Wards**: Kanyama, Chilenje, Mtendere, Rhodes Park, Mandevu, Chaisa, Chawama, Libala, Ng'ombe, Garden
- **Currency**: ZMW (Zambian Kwacha)
- **Date Format**: ISO 8601 (e.g., "2025-01-15T10:30:00Z")
